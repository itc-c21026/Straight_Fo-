## ゲームルール  
駒を掴み、スワイプで投げる  
カードは普通の駒や、様々な能力を持つ特殊駒を召喚できる  
カードを使うには、コストが必要で、コストは時間経過で増える  
お互いに駒を投げ合い、駒を4つ並べ、先にゲージを溜めたプレイヤーの勝利
